# Starbelly Examples

This directory contains runnable, production-relevant examples for Starbelly.

At the moment, this directory is intentionally minimal while example coverage is
rebuilt around supported API and protobuf features.

## Guidelines for adding examples

- Keep examples executable and up to date with the current server API.
- Include a short run command and expected output/behavior.
- Avoid placeholders that depend on unimplemented protocol fields.
